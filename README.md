# new-ara-dx
NewAra DX는 NewAra의 백엔드 개발자가 겪는 복잡한 환경설정을 모두 없애고, SPARCS 내부 서버에서 빠르게 본인의 개발 환경을 부여받을 수 있도록 도와주는 docker 이미지 및 compose 파일을 제공하고 있습니다.
